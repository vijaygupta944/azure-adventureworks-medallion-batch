# azure-adventureworks-medallion-batch
End-to-end batch Azure data engineering project using Medallion Architecture (ADF, ADLS, Databricks, Synapse, Power BI) with metadata-driven ingestion and lake-first analytics.

## Connectivity & Security
All service-to-service connections (Databricks–ADLS, Synapse–ADLS, Synapse–Power BI) are documented separately under the `connections/` folder following enterprise security best practices.
